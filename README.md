## 📌 Project Overview

The Smart Fitness Tracker Web App is a front-end web application designed to help users track their fitness journey.  
It allows users to log workouts, monitor progress, set goals, unlock achievements, and stay motivated.

---

## ⚙️ Features

### 🔹 Core Functionality

- **Workout Logging**
  - Log workouts with duration, calories, and date
  - View workout summaries and statistics

- **Statistics Dashboard**
  - Total workouts
  - Calories burned
  - Average workout duration
  - Charts using Chart.js

- **Fitness Goals**
  - Set personal fitness goals
  - Track progress with a progress bar
  - Motivational feedback

- **Achievements System**
  - Unlock badges based on activity
  - Stored using localStorage

- **Favourite Workouts**
  - Mark workouts as favourites ❤️
  - Dynamic UI updates

- **Authentication System**
  - Register and login functionality
  - Basic validation

## ⚡ How to Run the Project

1. Download or clone the repository:

```bash
git clone https://github.com/steven-riaan-piek/FitnessTrackerWebApp.git
```
2. Open the project folder
3. Open index.html in your browser

---

▶️ Usage
Log workouts on Log Workout page
View stats on Statistics page
Set goals on Goals page
Unlock achievements on Achievements page
Favourite workouts on Favourite page
